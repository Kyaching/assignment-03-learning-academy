Where gird was two dimensional layout approach (e.g row or column at the same time). A content in a flexbox shapes the layout. The box model in CSS is a set of rules that determine how your web page is rendered on the internet. In this model, a rectangular box is generated for HTML elements. Each is laid out according to their dimension, type, positioning, relationship to other elements, and external factors like viewport size. The box consists of content, padding, a border, and margin. There are several properties in the CSS box model. They are content area, padding area, border area and margin area.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.Semantic HTML tags let you serve structured content to your users, which is especially important for on-page SEO and accessibility. The following HTML tags can be used to break your page into identified parts: header, nav, section, article, footer etc.
